Css选择器

类选择器

.自定义类名 {属性: 值;}     类样的定义
◆类样的调用：
             标签通过class属性调用类样式
特点：
1 一个类样式可以同时被多个标签调用
2 一个标签可以同时调用多个类样式（人的名字）    
类的命名规范：
■不能使用数字或者以数字开头定义类名
■不推荐使用汉字定义类名
■不能使用特殊字符或者以特殊字符开头定义类名（“_”除外）

ID 选择器

#自定义名称 {属性: 值;}     样式的定义
ID样式的调用：
        标签通过ID属性调用定义好的样式即可。
ID选择器与类选择器之间的区别：
                1） 标签中的id值唯一 （身份证号码）
              2） 一个标签只能调用一个ID样式

通配符选择器

* {属性: 值; }
特点：该选择器可以将页面中的所有标签都选中。

后代选择器

  选择器 选择器{属性: 值;}
  注意：
  1） 后代选择器中，标签之间的关系必须是嵌套关系
  2） 后代选择器中，选择器与选择器之间使用空格隔开
3） 后代选择器中，能选中所有后代子元素

并集选择器

选择器,选择器,选择器 {属性: 值;}
 注意：
       1) 并集选择器中，标签之间可以是并列的关系也可以是嵌套关系。
     2）并集选择器中，选择器与选择器之间使用逗号隔开

       3）并集选择器中，可以选中所有满足条件的标签。

子代选择器

选择器>选择器 {属性: 值;}
注意：
        1) 子代选择器，标签之间必须是嵌套关系
          2）子代选择器，只能选中直接子元素

属性选择器

标签[属性] {属性：值;}
例：input[type='button']{
    color:red;
}

文本属性介绍

◆font-size     设置文字大小
◆font-weight  设置文字是否加粗显示
                 bold    文字加粗       与设置700一样
               normal  文字正常显示     与设置400一样
◆font-style   设置文字是否斜体显示
                 normal    文字正常显示
                 italic      文字斜体显示

◆font-family   设置文字字体
注意： 通过font-family可以设置多个字体，字体与字体之间使用逗号隔开。
◆line-height      文字行高

font属性联写

    font: font-weight   font-style   font-size/line-height    font-family;
注意：
          ◆font属性联写中，必须设置font-size 和font-family
          ◆font属性联写中，font-size 必须在font-family前设置
